# KeyScan Computer Vision System

## Overview
This directory contains the legacy computer vision-based KeyScan system that was used before the AI integration.

## Contents
- `keyscan.server.js` - Main computer vision service
- `keyscan-optimized.server.js` - Optimized computer vision service
- `vision/` - Computer vision algorithms and processors

## Features
- Traditional image processing
- Computer vision algorithms
- Shape matching
- Edge detection
- Pattern recognition

## Status
**DEPRECATED** - Replaced by AI-based system (V1 and V2)

## Versions
- V3, V4, V5 - Different iterations of computer vision system
- Legacy algorithms preserved for reference
